JavaScript and jQuery Flashcards

  • JavaScript (JS): a programming language used to make web pages interactive.
  • Operators: signs used to change the value of the variable
  • Variables: a container that stores information
  • Variable naming conventions: an agreed upon scheme for naming things
  • Functions: a bundle of code that performs a set of procedures
  • Methods: a kind of function that is called on something
  • Arguments: what you pass into a function or method
  • Parameters: a variable that is assigned to the argument
  • Return: a command to return a changed object in a method
  • Chaining Methods: when using a method on a method
  • Strings: object of compiled characters
  • Booleans: true or false objects
  • Undefined: objects without definition
  • NaN: not a number
  • Null: nothing
  • Alert: pop up message in browser
  • Comments: developer comments in code that are not run
  • jQuery: javascript library
  • Attributes: features of a program
